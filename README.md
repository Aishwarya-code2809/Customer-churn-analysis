# Customer Churn Analysis and Prediction

## 📌 Project Overview
This project analyzes customer churn in the telecom industry using machine learning and data visualization.

## 🚀 Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Random Forest model for prediction
- Power BI dashboard for insights

## 📊 Results
- Accuracy: 79%
- Key factors: tenure, monthly charges, contract type

## 📂 Files
- churn_analysis.ipynb → ML model
- dashboard.pbix → Power BI dashboard
- paper.pdf → Research paper

## 🛠️ Tools Used
- Python (Pandas, Sklearn)
- Power BI
- Jupyter Notebook

## 📈 Key Insight
Customers with low tenure and high charges are more likely to churn.
